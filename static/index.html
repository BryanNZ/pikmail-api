    Pikmail API

    Welcome to the Pikmail API! This is a pretty basic rest api written with Kort that uses the Pikmail library for mapping gmail addresses to google profile pictures.

    Advantages:

    • It's totally free and open source.
    • No authentication needed.
    • You don't need to write some wrapper for mapping the response.
    • It's written in Kotlin.

    This api is hosted on heroku as https://pikmail.herokuapp.com/ and it just exposes one endpoint.

    Profile

    Get Profile Picture

    curl -X GET 'https://pikmail.herokuapp.com/eduardo.alejandro.pool.ake@gmail.com?size=100' --verbose

    <img src="https://pikmail.herokuapp.com/eduardo.alejandro.pool.ake@gmail.com?size=100" alt="Profile Picture">

    Picasso.with(context).load("https://pikmail.herokuapp.com/eduardo.alejandro.pool.ake@gmail.com?size=100").into(imageView);

    This endpoint maps an Gmail address to a google profile picture url. <img src="https://pikmail.herokuapp.com/eduardo.alejandro.pool.ake@gmail.com?size=25" alt="Profile Picture"> will result on this once rendered.

    HTTP Request

    GET https://pikmail.herokuapp.com/{gmail}

    URL Parameters

    Parameter Description
    gmail The Gmail address of the profile to retrieve

    Query Parameters

    Parameter Required Default Description
    size Â NO If set, it defines the size of the profile picture, default google's size if omitted.
    If the Gmail address is found this endpoint redirects you to the Google profile picture url like this https://lh3.googleusercontent.com/-lNnqhGfaSJM/AAAAAAAAAAI/AAAAAAAAAAA/0cHYLCmUmIk/s64-c/101290539395397957916.jpg.

    Errors

    The Pikmail API uses the following error codes:

    Error Code Meaning
    302 Found -- The request will be redirected to the google profile picture url with the specified size if set.
    404 Not Found -- The specified Gmail account could not be found.
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
